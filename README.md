# Simple-React-Flow-Builder

## Clone Repository
```sh
git clone https://github.com/AbishekKumar-CSE/React-Flow-Builder.git
cd React-Flow-Builder
```

## Install Dependencies
```sh
npm install
# or
yarn install
```

## Run the Project
```sh
npm run dev
# or
yarn dev
```

Project will be available at `http://localhost:5173/`.
